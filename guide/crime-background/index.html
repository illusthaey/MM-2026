범죄경력조회 시스템(CRIMS) 이용방법 안내

·관련: 강원특별자치도원주교육지원청 행정과-32871(2025.9.2.)
·조회 대상자 본인이 CRIME 시스템을 통해 조회할 경우 당일 즉시 처리됩니다. ·최종 업데이트: 2026.3.15.(일) 오후 2:11

1. 로그인을 합니다

·관련: CRIMS 사용자 매뉴얼

  • 사이트 링크: https://crims.police.go.kr
    (※ 별도의 회원가입 불필요. 본인인증 후 바로 시스템 이용 가능)
  • 방법: 본인인증 후 시설 아이디와 검증번호 입력하고 조회 신청
  • 시설 아이디 및 검증번호는 고주무관에게 문의

2. 취업예정자 신청 5단계

·학교에서 취업예정자에게 그대로 안내해도 되는 흐름입니다.

1

시스템 접속 및 로그인

본인 명의 인증 방식으로 CRIMS에 접속합니다.

  • 간편인증, 공동인증서, 휴대전화, 아이핀 등으로 로그인합니다.
  • 별도 회원가입은 하지 않아도 됩니다.
2

시설(학교) 조회 및 동의

학교에서 받은 정보로 정확한 학교를 찾습니다.

  • 메인 화면에서 [취업예정자 발급동의 신청]을 선택합니다.
  • 시설 아이디(6자리)와 검증번호(4자리)를 입력하고 조회합니다.
  • 검색된 학교명이 맞는지 확인한 뒤 [동의]를 누릅니다.
3

발급 신청서 작성

회보서 유형과 인쇄 유형을 잘 고르는 것이 핵심입니다.

  • 학교에서 요구하는 회보서 유형을 선택합니다.
  • 직접출력: 본인이 직접 인쇄하여 제출할 때 선택합니다.
  • 시설(기관)출력: 학교가 시스템에서 직접 출력할 때 선택합니다.
  • 동의 사유, 전화번호, 주소, 관할 경찰서 등 상세 정보를 입력하고 신청합니다.
4

본인 범죄경력 확인

신청만으로 끝나지 않습니다. 반드시 본인 확인 절차가 필요합니다.

  • 경찰서 접수와 확인이 끝나면 상태가 회보서 발급대기로 바뀝니다.
  • [발급목록]에서 [본인 범죄경력 확인]을 눌러 내용을 먼저 확인합니다.
5

최종 발급 및 전송

선택한 인쇄 유형에 따라 마지막 버튼이 달라집니다.

직접출력 선택 시

  • 본인 범죄경력 확인 후 [직접출력]을 눌러 인쇄합니다.
  • 출력본을 학교에 직접 제출합니다.

시설(기관)출력 선택 시

  • 내용 확인 후 반드시 [본인확인완료(시설장출력)]을 눌러야 합니다.
  • 이 단계까지 끝나야 학교가 시스템에서 조회·인쇄할 수 있습니다.
가장 많이 놓치는 지점
학교에서 직접 출력할 예정인데도 취업예정자가 직접출력을 선택하거나, 본인확인완료(시설장출력) 버튼을 누르지 않으면 학교 쪽에서는 출력 권한이 생기지 않습니다.

3. 신청 화면 예시

·업로드된 CRIMS 사용자 매뉴얼 화면을 기준으로 핵심 장면만 추렸습니다.

신청 화면 예시 보기 (접기·펼치기)

① 취업예정자 발급동의 신청

메인 화면에서 신청 메뉴를 선택한 뒤 시설 아이디와 검증번호를 입력합니다.

② 회보서 유형 / 인쇄 유형 선택

회보서 종류를 고르고, 학교가 직접 출력할지 여부에 따라 인쇄 유형을 선택합니다.

③ 본인 범죄경력 확인

상태가 회보서 발급대기로 바뀌면 본인 범죄경력을 확인합니다.

④ 학교 운영자 / 업무담당자 인쇄

시설장출력으로 넘어온 건은 학교 운영자 또는 업무담당자가 발급목록에서 인쇄합니다.

4. 인쇄 유형 차이와 상태값

구분 누가 출력하나 취업예정자가 마지막에 해야 할 일 학교에서 확인할 점
직접출력 취업예정자 본인 본인 범죄경력 확인 후 직접출력 출력본을 별도로 제출받아야 함
시설(기관)출력 학교 운영자 또는 업무담당자 본인 범죄경력 확인 후 본인확인완료(시설장출력) 클릭 버튼이 안 보이면 본인확인완료 여부부터 확인

주요 상태값 이해

  • 회보서 발급대기: 본인 범죄경력 확인 단계로 넘어갈 수 있는 상태
  • 본인확인대기: 시설장출력을 선택했지만 아직 최종 확인을 끝내지 않은 상태
  • 발급완료(재인쇄 기간): 발급 후 재인쇄 가능한 기간
  • 처리 완료 / 신청 취소: 같은 건 재신청은 가능해지지만 기존 재인쇄는 막힐 수 있음
한 줄 정리
학교가 시스템에서 직접 출력하려면 취업예정자가 신청 단계에서 시설(기관)출력을 고르고, 발급대기 이후 본인확인완료(시설장출력)까지 눌러야 합니다.

5. 학교 운영자 / 업무담당자 출력 방법

학교 쪽 출력 순서

  1. CRIMS에 운영자 또는 업무담당자로 로그인합니다.
  2. [발급목록]으로 이동합니다.
  3. [취업예정자 발급동의 내역]에서 대상자를 찾습니다.
  4. 대상자 상태와 본인확인 완료 여부를 확인합니다.
  5. [인쇄] 버튼으로 회보서를 출력합니다.
재인쇄
발급 완료 후 5일간은 횟수 제한 없이 재인쇄할 수 있습니다.
인쇄 버튼이 안 보일 때
취업예정자가 본인확인완료(시설장출력)를 눌렀는지 먼저 확인하세요.

학교 담당자 체크리스트

  • 시설 등록 상태를 먼저 확인했다.
  • 시설 아이디 6자리와 검증번호 4자리를 정확히 전달했다.
  • 학교 직접 출력 예정이면 “시설(기관)출력”을 선택하도록 미리 안내했다.
  • 취업예정자가 “본인확인완료(시설장출력)”까지 눌렀는지 확인했다.
  • 출력 후 보관 및 제출 절차를 학교 내부 기준에 맞게 처리했다.

6. 자주 막히는 경우 / 주의사항

신규 신청이 안 되는 경우

  • 이전 신청 건이 발급대기 상태면 새 신청이 막힐 수 있습니다.
  • 이전 신청 건이 발급완료(재인쇄 기간) 상태여도 새 신청이 막힐 수 있습니다.
  • 필요하면 기존 건을 처리 완료 또는 신청 취소 후 다시 진행합니다.

처리 완료 / 신청 취소 전에 확인

  • 기존 건을 닫아 버리면 재인쇄가 불가능할 수 있습니다.
  • 학교 제출이나 출력이 끝났는지 먼저 확인한 뒤 처리하는 편이 안전합니다.

인쇄 유형을 잘못 선택한 경우

  • 학교가 출력해야 하는데 직접출력을 골랐다면 학교 쪽 인쇄가 되지 않습니다.
  • 안내 단계에서 “학교에서 출력 예정이면 시설(기관)출력” 문구를 꼭 넣어 주세요.

학교 검색이 안 되는 경우

  • 시설 등록이 아직 승인되지 않았을 수 있습니다.
  • 시설 아이디 6자리 또는 검증번호 4자리가 잘못 입력되었는지 확인합니다.
  • 학교장 변경이나 신규 등록 건이면 관련 공문 처리 상태도 함께 확인합니다.

학교에서 취업예정자에게 보내는 안내문 예시
