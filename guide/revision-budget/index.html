과목경정 요구하는 방법

·관련: 강원특별자치도 공립학교 회계규칙, 강원특별자치도교육비특별회계 재무회계규칙, 2026년도 학교회계 예산편성 기본지침, 2026 학교행정 업무편람
·최종 업데이트: 2026.03.06
·용도: 유치원·학교 실무자가 과목경정 요구 문안을 바로 작성할 수 있도록 정리한 안내 페이지

요약 및 목차

  • 대상: 유치원·학교에서 이미 집행한 지출의 예산 항목을 바로잡아야 하는 교직원
  • 핵심: 추경은 예산서 자체를 바꾸는 절차, 과목경정은 이미 집행된 건의 예산 항목을 바로잡는 절차입니다.
  • 실무 포인트: 과목경정 요구서는 보통 관련 문서번호 + 경정금액 + 경정사유 + 경정전/경정후 항목을 정확히 적으면 됩니다.

페이지 목차

1. 과목경정이란 2. 추가경정과 차이 3. 요구서 작성 순서 4. 실무 흐름·주의사항 5. 복사해서 쓰는 예시

1. 과목경정이란

·관련: 강원특별자치도 공립학교 회계규칙 제52조(준용), 강원특별자치도교육비특별회계 재무회계규칙, 학교행정 업무편람

실무에서 이해하면 쉬운 뜻

  • 과목경정은 이미 지출이 끝난 건의 예산 항목이 잘못 잡혀 있을 때, 그 항목을 사후에 바로잡는 절차입니다.
  • 쉽게 말하면 돈을 새로 더 편성하는 절차가 아니라, 이미 쓴 돈의 꼬리표를 바로잡는 절차입니다.
  • 학교회계에서 따로 정하지 않은 사항은 교육비특별회계 재무회계규칙을 준용하므로, 과목경정도 그 기준에 따라 이해하면 됩니다.

이럴 때 과목경정을 검토합니다

품의 시 항목 착오 원인행위 시 항목 착오 지출은 이미 완료 같은 회계연도 안
  • 원래는 A 세부항목에서 나가야 하는데, 실수로 B 세부항목에서 집행한 경우
  • 본예산·추경 편성은 맞지만, 실제 지출결의에서 잘못된 산출내역을 선택한 경우
  • 우선 다른 항목에서 집행했고, 나중에 적정 항목으로 정리해야 하는 경우
과목경정 전에 먼저 확인할 것
  • 첫째 같은 회계연도 안의 건인지 확인합니다.
  • 둘째 해당 지출이 이미 완료되었는지, 특히 이체확정이 끝났는지 확인합니다.
  • 셋째 경정전·경정후 예산 경로를 정책사업/단위사업/세부사업/세부항목까지 정확히 적어둡니다.
  • 넷째 관련 품의 문서번호, 지출결의 내역, 증빙서류를 같이 묶어두면 이후 편철이 편합니다.

2. 추가경정과 과목경정의 차이

·관련: 2026년도 학교회계 예산편성 기본지침, 2026 학교행정 업무편람

한 문장으로 정리

추가경정은 “예산서 자체를 고치는 것”, 과목경정은 “이미 지출된 건의 예산 항목을 바로잡는 것”입니다.

구분 추가경정(추경) 과목경정
대상 이미 성립된 예산서 이미 집행된 지출 항목
언제 하나 예산의 목적이 바뀌거나, 기정예산을 변경할 필요가 생겼을 때 지출 후 회계명·세출과목 등에 착오가 있어 바로잡아야 할 때
심의 여부 학교운영위원회 심의 필요 내부결재로 처리
예산액 변화 예산액 자체가 변동될 수 있음 예산액을 새로 늘리는 절차는 아님
실무 기억법 계획을 고친다 집행된 건의 항목을 고친다

3. 과목경정 요구서 작성 순서

문안 구성은 보통 이렇게 갑니다

  • 1)제목: 연도 + 월 + 사업명 + 금액/항목 + “과목경정 요구”
  • 2)관련: 원래 집행한 품의 문서번호
  • 3)경정금액: 실제 바로잡을 금액
  • 4)경정사유: 예산 착오 사용, 항목 착오 집행 등 짧고 명확하게
  • 5)경정내역: 경정전 / 경정후 항목을 모두 적기
  • 6)결재 흐름: 기안 후 행정실 공람 또는 내부결재 라인에 태우기
항목 어떻게 쓰면 좋은지 작성 팁
제목 "2025년 9월 유치원 특수교육대상학생 교육지원 자원봉사자 활동비 과목경정 요구" 사업명을 너무 줄이지 말고, 나중에 찾아보기 쉽게 씁니다.
관련 원래 품의한 문서번호 기재 문막초등학교-12971(2025.10.2.)처럼 날짜까지 같이 쓰면 좋습니다.
경정금액 금418,000원(금사십일만팔천원) 아라비아 숫자와 한글 금액을 같이 적으면 문안이 깔끔합니다.
경정사유 예산 착오 사용 길게 쓰기보다 원인만 짧게 적는 편이 실무상 읽기 쉽습니다.
경정내역 경정전 / 경정후 예산 경로 전부 기재 세부사업, 세부항목, 원가비목, 차수(2차/3차)까지 정확히 맞춰야 합니다.

(※ 핵심은 “왜 바꾸는지”보다 “어디서 어디로 바꾸는지”를 명확히 쓰는 것입니다.)

제목을 쉽게 만드는 공식

[연도] + [집행 시기] + [사업명] + [지출 성격] + 과목경정 요구

  • 예: 2025년 9월 유치원 특수교육대상학생 교육지원 자원봉사자 활동비 과목경정 요구
  • 예: 2026학년도 1학기 현장체험학습 차량임차료 과목경정 요구

4. 실무 흐름과 주의사항

실무 흐름은 보통 이렇게 이해하면 됩니다

  • 품의 또는 원인행위 시 예산 항목을 잘못 설정한 경우, 지출이 끝난 뒤 과목경정을 진행합니다.
  • 보통 이체확정 이후 전체경정 또는 분할경정을 진행합니다.
  • 금액 전체를 다른 산출내역으로 옮기면 전체경정, 일부 금액만 옮기면 분할경정입니다.

주의할 점

  • 분할지출결의, 반납결의, 기과목경정이 이미 있는 건은 분할경정이 안 될 수 있습니다.
  • 예산편성 단계에서 세부사업·원가통계비목을 잘못 잡아 과목경정이 반복되는 일은 지양하는 것이 좋습니다.
  • 지출증빙 편철 시에는 과목경정 관계서류도 함께 묶어 두어야 나중에 결산·감사 때 설명이 쉽습니다.

헷갈리면 이렇게 판단하세요

  • 지출 전이고 예산 계획 자체를 바꿔야 하면 → 추가경정 검토
  • 지출 후이고 항목만 잘못 들어갔으면 → 과목경정 검토
  • 같은 정책사업 안에서 아직 집행 전 조정이 가능한 경우 → 이용·전용 가능 여부도 함께 검토
