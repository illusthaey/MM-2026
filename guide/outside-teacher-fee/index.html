·최종 업데이트: 2026.03.01

핵심 요약

  • 대상: 문막초 교직원(공무원·교원·교육공무직·학교장 임용 직원)
  • 적용: 초과근무 발생 시(조기출근/정규근무 이후/토·일·공휴일 등)
  • 해야 할 일(3단계):
    ① 결재(명령) 완료 → ② 근무지에 맞는 증빙 확보(교내=지문 / 교외=수기대장) → ③ 신분별 시간 산정(공무원=평일 1시간 공제 / 근로기준법 적용자=휴게시간 공제)

안내 내용

빠른 이동

0) 3단계 체크(결재→근무지→증빙) 1) NEIS(나이스) 신청·완결 확인 2) 근무지별 증빙(교내/교외) 3) 공무원(교원·지방공무원) 산정·한도 4) 근로기준법 적용자(공무직 등) 산정 5) 자주 묻는 질문(사례 포함)

0) 초과근무 3단계 체크 (결재 → 근무지 → 증빙)

가장 흔한 미인정 사유는 “결재 누락” 또는 “증빙 누락(지문 한쪽만)”입니다.
원칙은 사전 신청·승인(명령)이며, 부득이한 경우에도 퇴청 시 당직 확인 + 다음 날까지 사후결재가 필요합니다.

초과근무 발생 시 체크리스트(흐름도)

  • 1단계(결재): 초과근무 복무 상신이 “완결” 되었나요?
  • 2단계(근무지): 근무지가 학교 안인가요, 학교 밖인가요?
  • 3단계(증빙):
    - 교내: 초과근무 구간의 출근·퇴근 지문 모두 필요
    - 교외: 초과근무 확인 수기대장 기재(출장과 별도로 초과근무 결재 필요)

1) NEIS(나이스) 초과근무 신청·완결 확인

·초과근무는 “신청 → 승인(명령) → 확인 → 월별집계(급여반영)” 흐름으로 관리됩니다.

단계 무엇을 하나요 NEIS 경로(요약)
신청(상신) ·초과근무 일시/사유/해야 할 일(구체 업무)을 입력하고 승인 요청
·승인자 지정 후 결재 상신 → 기결함에서 “완결” 확인 		복무 → 개인초과근무관리 → 신청
확인(증빙) ·근무지에 맞는 증빙(지문/수기대장)으로 실제 근무 사실을 확인
·복무담당이 초과근무확인에 시간 입력/마감(NEIS 활용) 		복무 → 초과근무관리 → 초과근무확인
월별집계 ·다음 달 급여 반영을 위해 월별 집계/생성 및 저장(정액분 포함) 복무 → 초과근무관리 → 초과근무월별집계

(※ “사전 승인”은 초과근무 “명령”으로 간주됩니다. 사전 명령 없이 근무했거나 명령시간을 초과한 경우, 퇴청 시 당직 확인 후 다음 날까지 사후결재가 필요할 수 있습니다.)

2) 근무지별 증빙 방법 (교내/교외)

교내 초과근무는 지문 “출근·퇴근” 둘 다 있어야 인정됩니다.
정규근무와 초과근무 사이에 공백이 없더라도, 초과근무 구간의 지문이 누락되면 “증명 불가”로 미인정될 수 있습니다.
구분 증빙자료 확인수기대장 주의사항
교내 근무 지문 출퇴근 내역(초과근무 구간의 출근·퇴근 모두) 기재 X ·정규근무(예: 8:40~16:40)와 초과근무는 별개
·출근만/퇴근만 있는 경우 “근무 시간”으로 인정 불가 사례가 많습니다
교외 근무 초과근무 확인 수기대장 기재(필요 시 활동 증빙 첨부) 기재 O ·출장과 별도로 초과근무 결재(명령)가 있어야 “기재 근거”가 생깁니다
·출장 중 초과근무는 원칙적으로 인정되지 않으나, 목적상 필연적으로 발생이 예상되는 경우 명령에 따라 지급 가능(사안별 확인)

(※ 교외 근무는 상황이 다양하므로, “결재(명령) + 수기대장”을 기본으로 생각하시면 안전합니다.)

3) 공무원(교원·지방공무원) 초과근무: 인정/산정/한도

핵심 포인트

  • 한도: 1일 4시간, 월 57시간 이내(근무명령시간 상한)
  • 평일 산정: 정규근무 전/후 시간외근무는 1일 1시간 공제 후 매분 단위 합산(공제 후 1시간 미만이면 미지급)
  • 휴무토/휴일: 1시간 이상이면 공제 없이 매분 단위 인정(단, 1일 4시간 상한)
  • 정액분: 월간 출근근무일수 15일 이상이면 월 10시간분 정액 지급(조건 미달 시 일할 계산)
  • 대체휴무: 토·일·공휴일에 연속 8시간 이상 근무한 경우, 수당 대신 다음 정상근무일(또는 6주 이내 지정일) 휴무 부여 가능
  • 중복 지급 금지: 동일 업무에 대해 시험감시·보충수업 지도 등 다른 방식의 금전 보상을 받는 경우, 해당 시간외근무수당 지급 제외 대상이 될 수 있음
항목 기준(요약) 예시/비고
지급시간 상한 1일 4시간 / 월 57시간 이내 근무명령 시간 기준 상한
평일(정규근무 전/후) 1일 1시간 공제 후 매분 단위 합산(공제 후 1시간 미만: 미지급) 조기출근 + 퇴근 후 근무는 합산 후 공제
휴무토/휴일 1시간 이상이면 공제 없이 매분 단위 인정(1일 4시간 상한) 1시간 미만: 미지급
정액분(월 10시간) 출근근무일수 15일 이상: 10시간 정액 / 15일 미만: (출근근무일수/20)*10 정액분은 별도 명령 없이 지급 가능(조건 충족 시)
대체휴무 토·일·공휴일에 연속 8시간 이상 근무 시 다음 정상근무일 휴무(또는 6주 이내 지정) 기관 사정에 따라 조정 가능

(※ 세부 해석/예외는 업무편람 및 복무규정, 그리고 학교 내부 운영 기준을 따릅니다.)

4) 교육공무직·학교장 임용 직원(근로기준법 적용자) 초과근무: 휴게시간/연장근로
핵심: 근로기준법 적용자는 휴게시간을 공제하여 근무시간을 산정합니다.
4시간 이상 근무 시 최소 30분, 8시간 이상 근무 시 최소 1시간 휴게가 필요하며, 무급 휴게시간을 제외하고 NEIS에 상신합니다.
항목 기준(요약) 비고
법정근로시간 1일 8시간, 1주 40시간(휴게시간 제외) 연장근로·휴일근로는 별도 기준 적용
연장근로 한도 당사자 합의로 1주 12시간 한도 강제 지시는 불가(합의 필요)
야간근로 22:00~06:00 사이의 근로 야간 가산수당 대상
가산수당 연장·야간·휴일근로는 통상임금의 50% 이상 가산 휴일 중첩 등은 규정에 따라 달라질 수 있음
휴게시간 반영 NEIS 상신 시 휴게시간을 제외하고 시간 수정(무급 휴게 공제) 학교별 직종별 휴게시간은 별도 문서(문막초-4064) 참고

(※ 근로시간/휴게시간은 “직종별”로 정해진 기준이 있으니, 학교 내부 문서(문막초-4064)로 본인 직종 기준을 확인하세요.)

5) 자주 묻는 질문(FAQ) · 실제 사례

Q1. 교내 초과근무인데 “출근 지문만” 또는 “퇴근 지문만” 찍었어요.
인정 불가(증명 불가) 사례가 매우 많습니다.
정규근무와 공백이 없고 실제로 계속 근무했더라도, 초과근무 구간의 출·퇴근 지문이 모두 있어야 “초과근무 종료 시각”을 증명할 수 있습니다.
  • 조기출근 초과근무 후 정규근무를 계속한 경우라도, 초과근무 종료(예: 08:40) 지문이 없으면 불인정될 수 있습니다.
  • 정규근무 후 초과근무를 이어서 한 경우라도, 초과근무 시작(예: 16:40) 지문이 없으면 불인정될 수 있습니다.
Q2. 교외(학교 밖) 초과근무는 어떻게 증빙하나요?
원칙: 교외 근무는 초과근무 확인 수기대장 기재가 기본입니다.
중요: 출장과 별도로 “초과근무 결재(명령)”가 있어야 수당 지급 근거가 됩니다.
  • 현장체험학습, 대회·행사 인솔, 외부 교육/연수 등으로 학교 밖에서 초과근무가 발생하면 수기대장에 기재합니다.
  • 출장 중 초과근무는 원칙적으로 인정되지 않으나, 목적상 필연적으로 발생이 예상되는 경우 시간외근무명령에 따라 지급 가능한 예외가 있을 수 있습니다(사전 상담 권장).
Q3. 결재는 했는데, 왜 “일부만 인정”되나요?

·초과근무는 결재(명령) + 증빙(교내 지문/교외 수기대장)이 함께 충족되어야 인정됩니다.

사람 사전결재 출근 지문 퇴근 지문 결과(예시)
A O O O 인정 (결재+증빙 충족)
B O O X 미인정 (증빙 누락)
C X O O 미인정 (결재 누락)

(※ 실제 사례에서도 “동일 사유”라도 조건 미충족 시 미인정된 경우가 있습니다.)

예시 화면(개인정보가 포함될 수 있으니 게시용 업로드 전 가림처리 권장)

Q4. 교육공무직(근로기준법 적용자)도 “평일 1시간 공제”하나요?
교육공무직은 공무원 방식의 “일괄 1시간 공제”가 아니라, 실제 휴게시간을 공제하여 산정합니다.
  • 4시간 이상 근무 시 최소 30분, 8시간 이상 근무 시 최소 1시간 휴게시간을 고려해 NEIS 상신 시간을 조정합니다.
  • 직종별 고정 휴게시간이 있는 경우, 학교 문서(문막초-4064)를 기준으로 합니다.
문의: 결재/증빙/시간 산정이 애매할 때는 “상신 전에” 행정실(복무 담당)로 먼저 상담하면 재상신/미인정 리스크를 크게 줄일 수 있습니다.
