공문서 생산

기안 → 검토·협조 → 결재 → 등록 → 시행 순서로 보면 됩니다.

  • 제목은 짧고 명확하게 쓰고, 본문은 필요한 항목만 남겨 길이를 줄입니다.
  • 내부결재 문서는 발신명의를 쓰지 않고, 대외문서는 결재 후 시행까지 이어집니다.
  • 수정이 생기면 원칙적으로 다시 결재를 받아 최종본 기준으로 시행합니다.

공문서 접수

접수번호를 먼저 찍고, 내 업무가 아니면 반송·이송 판단을 바로 합니다.

  • 접수 즉시 등록하고 담당자에게 배부합니다.
  • 소관이 아니면 발신기관으로 반송하거나 소관기관으로 이송합니다.
  • 민원·협조·검토가 필요한 문서는 공람 순서를 정해 놓으면 실수가 줄어듭니다.
접수 배부 반송·이송

기록물 관리

등록 → 편철 → 이관 → 생산현황 통보 → 보존 → 평가·폐기 흐름으로 관리합니다.

  • 전자문서는 시스템에서 자동 등록되고, 비전자문서는 별도 등록이 필요합니다.
  • 업무가 끝나면 표지·색인목록을 붙여 편철하고 보존상자에 정리합니다.
  • 학교에서 임의 폐기하지 말고, 평가·폐기 절차는 기록관리 체계에 맞춰 처리합니다.
등록 편철 이관 폐기 금지

사무인계·직무대리

발령 직후에는 장부 마감, 현황 정리, 인계서 작성, 직무대리 설정을 먼저 챙깁니다.

  • 학교장·행정실장·출납원 변경 시 장부와 현황표를 같이 넘겨야 빠지는 일이 없습니다.
  • 직무대리는 법정대리와 지정대리로 나뉘므로, 공문 명의와 결재 표시를 구분합니다.
  • 회계관계공무원 직무대리는 시스템 권한 변경까지 같이 처리해야 실제 업무가 굴러갑니다.
인계서 장부 마감 직무대리

한입 메모: 문서는 “만드는 것”보다 “등록·보존까지 닫는 것”이 실무입니다. 접수번호, 생산등록번호, 편철 상태를 마지막에 한 번 더 확인하면 사고가 크게 줄어듭니다.

휴가 구분

연가·병가·공가·특별휴가를 먼저 정확히 구분해야 결근 처리나 연가 공제를 막을 수 있습니다.

  • 병가는 진단서 제출 기준과 연가 전환 기준을 같이 봐야 합니다.
  • 특별휴가는 경조사, 출산, 유산·사산, 난임치료, 육아시간, 가족돌봄 등으로 나뉩니다.
  • 연가일수는 재직기간, 사실상 직무 미종사 기간, 전년도 가산여부까지 같이 계산합니다.
연가 병가 공가 특별휴가

출장

출장은 공무 목적, 장소, 시간, 결과보고가 남아야 여비와 복무가 같이 정리됩니다.

  • 근무지내·근무지외를 구분하고, 출장 용무는 최대한 구체적으로 적습니다.
  • 출장이 끝나면 결과보고 또는 필요한 후속조치를 바로 남겨야 합니다.
  • 국외출장은 심사, 허가, 귀국 후 보고서 공개까지 별도 흐름으로 봐야 합니다.
국내출장 공무국외출장 결과보고

초과근무·당직

사전 신청·승인, 실제 확인, 월 집계, 수당 반영까지 닫아야 합니다.

  • 초과근무는 개인별 구체 업무가 적힌 명령이 있어야 하고, 사후결재는 예외적으로만 처리합니다.
  • 연속 8시간 이상 근무 등은 대체휴무 검토 대상이 될 수 있습니다.
  • 당직은 보안점검, 긴급연락, 비상소집체계 확인이 핵심이고, 재택당직 여부는 학교 운영계획에 따릅니다.
시간외근무 대체휴무 당직 비상근무

한입 메모: 복무는 “사유가 맞는지”보다 “증빙과 시스템 반영이 같이 됐는지”에서 사고가 많이 납니다. 특히 병가, 출장, 초과근무는 사후 입력 누락을 제일 먼저 점검하세요.

계약제교원 채용, 구비서류, 조회, 보고

계약제교원은 “채용 방식 → 서류 → 조회 → 계약 → 보고 → 인사기록” 순서가 틀어지지 않게 보는 것이 핵심입니다.

채용 흐름

결원 발생 → 임용계획 수립 → 인력풀 우선 → 필요 시 공고 → 심사 → 계약·임용으로 보면 됩니다.

  • 기간제교원은 인력풀을 우선 활용하고, 없으면 공개채용으로 넘어갑니다.
  • 심사위원회 구성, 면접·수업실연 기준, 명예퇴직자 후순위 여부를 채용 전에 정리해 둡니다.
  • 계약은 표준계약서를 기본으로 하고, 구두 약속이 남지 않게 문서로 닫습니다.
인력풀 공개채용 표준계약서

구비서류·조회

서류는 최소한으로 받고, 필수 조회는 빠짐없이 남겨야 합니다.

  • 자격증, 학력, 경력, 행정정보 공동이용 동의, 개인정보 제공 동의 등 기본 서류를 확인합니다.
  • 결격사유, 성범죄·아동학대, 장애인학대(해당 시), 마약류 검사, 가족 채용 제한 여부를 같이 점검합니다.
  • 특수학교·특수학급·특수교육지원센터는 장애인학대 관련 조회를 따로 챙깁니다.
결격조회 범죄경력조회 마약류 검사 가족 채용 제한

복무·보험·보고

임용이 끝나면 NEIS 등록, 임용보고, 4대보험, 연수·평가 관리까지 이어집니다.

  • 기간제교원과 정규교원 대체 강사는 정해진 기한 안에 임용보고를 해야 합니다.
  • 4대보험은 기간·근로시간 조건에 따라 달라지므로 강사의 적용기준을 별도로 확인합니다.
  • 2개월 이상 근무평가, 6개월 이상 연수 관리, 육아휴직 허용 여부와 계약종료 시점을 같이 봅니다.
임용보고 4대보험 육아휴직 평가·연수

한입 메모: 계약제교원 업무는 “채용”보다 “조회와 기록”에서 감사 포인트가 많이 잡힙니다. 계약서, 조회 회보, 임용보고, NEIS 입력이 한 세트로 닫혔는지 보세요.

학교회계 예산·결산 예산편성, 성립전예산, 추경, 이·전용, 결산

예산편성

예산편성기본지침 확인 → 의견수렴 → 예산안 편성 → 운영위 제출 → 확정·공개 순서입니다.

  • 학생·학부모·교직원 의견수렴과 예산요구서 반영 여부를 먼저 정리합니다.
  • 전입금 통보 이후 가용재원 범위에서 예산을 조정하고 운영위원회 심의로 갑니다.
  • 확정된 예산은 홈페이지 등에 공개해 투명성을 확보합니다.

성립전예산·추경·간주처리

용도 지정과 전액 교부 여부, 운영위 심의 가능 시점, 차기 추경 계상 여부를 같이 봅니다.

  • 성립전예산은 용도가 지정되고 소요 전액이 교부된 경비 등에서만 제한적으로 사용합니다.
  • 추경은 예산 성립 후 새 사유가 생겨 기존 예산을 바꿔야 할 때 편성합니다.
  • 간주처리는 회계연도 말 목적지정 사업비 등 운영위 심의 시간 여유가 없을 때 제한적으로 씁니다.

예산 이용·전용·이월

목적 외 사용 금지, 이용·전용 가능 범위, 이월의 종류와 제한을 같이 봐야 합니다.

  • 업무추진비 충당, 신규사업 추진, 이월예산 전용 등은 제한사항을 먼저 확인합니다.
  • 이용은 정책사업 간, 전용은 같은 정책사업 안에서의 조정으로 구분해 봅니다.
  • 명시이월·사고이월·계속비이월은 사유와 절차가 서로 다르므로 구분해서 처리합니다.

결산·공개·반납

결산서는 작성 시점, 첨부서류, 운영위 심의, 공개, 반납까지 한 묶음으로 봅니다.

  • 예비비 사용, 이용·전용 내역, 이월경비 내역 등 첨부서류를 같이 정리합니다.
  • 결산 심의 후에는 홈페이지 등에 공개하고, 반환금·순세계잉여금 정리도 놓치지 않습니다.
  • 목적사업비 집행잔액과 학교운영비 반납 기준은 연말에 따로 체크리스트로 보는 것이 안전합니다.
결산 반환금 예결산 공개

한입 메모: 학교회계는 “편성”보다 “예외 처리”에서 실수가 납니다. 성립전예산, 간주처리, 전용, 반환금은 각각 별도 체크리스트로 끊어서 보세요.

지출·계약·여비 증빙, 업무추진비, 수의계약, 여비

지출은 “증빙 → 예산과목 → 집행방법 → 공개”를 같은 화면에서 같이 보는 것이 안전합니다.

지출증빙 기본

결의서, 영수증서, 계약증빙이 한 묶음으로 남아야 지출이 닫힙니다.

  • 지출원인행위와 지급청구의 정당성을 먼저 확인한 뒤 지출합니다.
  • 계좌입금 증빙, 지출결의서, 계약 관련 증빙을 월별 또는 분기별로 편철합니다.
  • 소액계약, 반납 처리, 세출과목 복귀 여부를 같이 보면 결산 때 편합니다.

업무추진비·특근매식비

허용 범위, 증빙 방식, 공개 기준을 먼저 잡고 써야 안전합니다.
