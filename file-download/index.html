⚠️ 브라우저/모바일 환경에 따라 PDF는 ‘다운로드’가 아니라 ‘열기’로 보일 수 있습니다. 그럴 땐 PDF 뷰어의 저장(⬇︎) 버튼을 사용하세요.

업로드/추가 방법(관리자용)

  1. 파일 업로드: 서버(또는 저장소)에 파일을 올립니다. 권장 경로: /files/forms/
  2. 파일명 정리: 한글/특수문자 파일명은 서버 환경에 따라 링크가 깨질 수 있어요. 가능하면 영문/숫자/하이픈으로 변경을 권장합니다.
  3. 목록 등록: 이 HTML 파일 하단의 FILES 배열에 항목 한 줄 추가하면 끝.
  4. ZIP 묶음(선택): 자주 쓰는 파일은 ZIP으로 묶어 /files/forms/eduworkhaey-forms-pack.zip로 올리면 ‘전체 묶음’ 버튼이 동작합니다.

FILES 항목 추가 예시

{
  title: "출장신청서(서식)",
  format: "HWP",
  size: "92KB",
  updated: "2026-03-02",
  path: "/files/forms/travel-request-form.hwp",
  originalName: "출장신청서.hwp",
  desc: "관내/관외 출장 신청서 서식(학교 자체 양식).",
  tags: ["출장", "복무", "서식"]
}
