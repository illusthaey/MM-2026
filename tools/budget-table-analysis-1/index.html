사업관리카드(담당) 마이너스 예산잔액 분석기

사업관리카드(담당) 형식의 엑셀을 브라우저에서 바로 분석합니다. 표시할 열과 틀고정 열을 선택할 수 있고, 음수 검출 기준은 F열(원인행위 후 예산 잔액)입니다.

  • 사업관리카드(담당)처럼 제목 행이 위에 있고 헤더가 1줄인 표를 기준으로 동작합니다. 헤더 행은 자동 감지되며 필요하면 직접 바꿀 수 있습니다.
  • 병합 셀이 있으면 기본으로 값을 펼쳐서 보여주고, 필요하면 해제해서 원본처럼 볼 수 있습니다.
  • 음수 검출 기준은 항상 F열입니다. 표에서 F열을 숨겨도 검출 목록과 행 이동은 유지됩니다.
  • 결과 다운로드는 현재 화면 기준 음수 항목 목록을 별도 엑셀 파일로 저장합니다.
