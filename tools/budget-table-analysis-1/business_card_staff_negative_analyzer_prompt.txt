다음 기존 HTML/JS 스타일을 참고해서, **사업관리카드(담당) 마이너스 예산잔액 분석기**를 만드는 단일 HTML 소스코드를 작성해줘.

[핵심 목적]
- 사업관리카드(담당) 엑셀 파일을 브라우저에서 바로 읽는다.
- **F열을 음수 검출 기준 열**로 사용한다.
- F열 값이 0보다 작으면 음수 항목으로 검출하고, 표와 검출 목록에서 강조한다.
- 사용자는 **표시할 열 선택**과 **틀고정 열 선택**을 할 수 있어야 한다.

[파일 형식]
- `.xls`, `.xlsx`, `.xlsm` 업로드 지원
- 드래그앤드롭 + 클릭 업로드 둘 다 지원
- 파일은 서버 업로드 없이 브라우저에서만 처리

[사업관리카드(담당) 구조 가정]
- 샘플 기준 헤더는 보통 3행이지만, 자동 감지 후 사용자가 헤더 행 번호를 직접 수정할 수 있게 해줘.
- 헤더 위쪽 행들은 “헤더 위쪽 정보” 카드로 따로 보여줘.
- 병합 셀이 있으면 기본값은 “값 펼쳐서 보기” 켜짐으로 처리해줘.

[음수 검출 규칙]
- 기준 열은 **고정 F열(0-based index 5)** 로 처리
- F열 헤더명은 파일마다 조금 달라도, **열 위치 우선**으로 F열을 사용
- F열이 존재하지 않으면 “현재 시트에 F열이 없어 음수 검출 불가” 안내 표시
- 음수 검출 목록 카드에서:
  - 행번호
  - F열 헤더명
  - F열 값
  - A열/ B열 주요 식별 텍스트
  를 한 줄 요약으로 보여줘
- 목록 클릭 시 아래 표의 해당 행으로 부드럽게 이동하고 포커스 표시

[표 기능]
- 헤더 행 sticky
- 사용자가 선택한 개수만큼 왼쪽 열 sticky
- 표시 컬럼 체크박스 UI 제공
- “전체 표시”, “앞 4개만” 버튼 제공
- 검색창으로 행 필터링
- 표 hover / focus 스타일
- F열 음수 셀은 연한 빨강 배경 + 진한 빨강 글씨로 강조

[추가 기능]
- 현재 화면 기준 음수 항목만 별도 엑셀 파일로 다운로드하는 버튼 제공
- 다운로드 파일에는:
  - 원본시트
  - 원본행
  - 원본 전체 열
  이 포함되게 해줘

[디자인]
- 카드형 UI
- 업로드 영역, 제어영역, 헤더 위쪽 정보, 음수 검출 내역, 표 미리보기로 구분
- 모바일 반응형
- `body class="alien-side-bg"` 와 `.alien-page-frame` 구조 사용 가능
- 외부 공통 리소스는 아래처럼 참조 가능
  - `/static/style.css`
  - `/static/footer.js`
  - `/static/global-loader.js?v=1`

[기술 구현]
- 단일 HTML 파일
- 브라우저에서 동작하는 순수 JS
- old `.xls` 지원 때문에 SheetJS 로딩 로직 포함
- 가능하면 `/static/xlsx.full.min.js`를 먼저 시도하고, 없으면 CDN을 fallback으로 시도
- 코드 안에 주석 충분히 넣기
- 함수 분리 깔끔하게 하기
- 에러 메시지는 한국어로 친절하게 작성

[중요]
- 기존 “사업관리카드(현액)” 뷰어 느낌을 최대한 유지하되,
- **음수 검출 기준만 F열로 고정**
- **음수 컬럼 선택 UI는 빼고**, 표시 컬럼 선택 UI만 남겨줘
- 결과물은 바로 저장해서 배포 가능한 완성 HTML이어야 한다.
