3. 특별 지원 및 건강검진 안내

특별건강검진 (격년 배정)
  • 2026년 대상자: 짝수년도 출생자
  • 지원 한도: 300점(금300,000원) 한도 내 실지출 금액 지급 (월할계산 하지 않음)
  • 배정 대상: 휴직자 포함 배정
  • 사용 범위: 본인 검진 가능 / 본인 지출 시 가족 검진에도 사용 가능(배우자, 직계 존·비속 및 배우자의 직계존속)
  • 마감: 2026.12.09.(수)까지 조정점수 및 영수증 청구를 사용자 본인이 완료
진행 순서
  1. 검진
  2. 사용자: 포탈에서 조정점수 신청
  3. 담당자: 승인
  4. 사용자: 영수증 청구(증빙서류 첨부 필수)
출산·난임·태아·산모 검진 지원
  • 출산 축하 점수: 자녀당 1회(첫째 1,000점 / 둘째 3,000점 / 셋째 이상 5,000점)
  • 신청 기한: 출생 연도 포함 3년 이내
  • 난임 지원: 재직 중 1회, 500점
  • 태아·산모 검진: 자녀 수 1명당 100점

※ 증빙서류 제출 필요(지원청 현황 보고).
- 제출: 맞춤형 복지 신청서(서명 날인) + 증빙서류 일체

유의사항(승인/청구 흐름)
  1. 사용자: 사유 발생 후 증빙서류 첨부 → 조정점수 신청
  2. 담당자: 조정점수 승인 (여기까지 = 점수 배정 상태)
  3. 사용자: 포탈에서 영수증 청구 진행
  4. 담당자: 청구 내역 최종 승인 → 다음 달 급여일 지급

4. 단체보험 및 항목 선택

  • 필수 항목: 생명·상해 보장보험 의무 가입(보장금액 5천만/1억/2억 중 선택)
  • 선택 항목: 특정질병 진단비, 치아보험, 실손의료비 등은 개인 필요에 따라 선택
  • 실손보험 중복 확인: 개인 실손보험이 있으면 단체 실손보험 가입 중지 가능 → 중복 여부 확인

5. 신분 변동 시 유의사항

  • 월할 계산: 신규 임용/퇴직/휴직 등 변동 발생 시 실제 근무한 달 기준으로 점수 월할
  • 퇴직 예정자: 정년·명예퇴직 예정자는 퇴직 전 점수 사용 권장 (퇴직 후 2개월 이내에만 청구 가능)
  • 퇴직 후 마감 처리: 퇴직 후 2개월 경과 시 “퇴직 처리현황”에서 “정산중” → 팝업 하단 “정산완료”
  • 이월·현금화 불가: 당해 연도 미사용 점수는 다음 연도 이월/금전 보상 불가 → 기한 내 전액 사용

6. 결제 및 청구 방식

  • 카드 등록: 포탈에 복지카드 등록 후 자동청구 또는 수동청구 선택
  • 사용 제한: 사행성 지출, 유흥비, 미용 관련 의료행위(성형/치열교정 등), 일반 상품권 구매 등 제한
  • 간편결제 유의: PAYCO/네이버페이 사용자는 일부 항목(특별검진 등)에서 영수증 청구 불가 → 조정점수 부여 후 자율항목으로 사용
지급 시기 및 문의처

·지급 시기: 승인일의 다음 달 급여일 지급 (예: 2월 승인 → 3월 급여일 지급)
·지급 문의: 교육공무원은 유초등교육과 / 지방공무원은 도교육청 총무과 문의
