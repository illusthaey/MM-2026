필수 확인 사항

·맞춤형복지 포털

  • 포털 로그인 후 복지점수(기본/근속/가족)를 본인이 직접 확인
  • 오류 있으면 즉시 담당자에게 변경 요청
  • 지급 계좌가 본인 명의인지 확인
  • 자율항목(보험료 제외) 금액의 10%는 전통시장(온누리) 또는 지역사랑 상품권 포인트로 자동 전환
  • 마감(2026.12.09.) 이후 신청/지급 불가 → 미사용 점수는 이월·현금화 불가
헷갈림 TOP 3 (꼭 읽어주세요)
  • 전출/전보 처리 시점: 인사발령 “이전” 선처리 금지. 3.1.자 전출자는 3.1. 이후 전출처리가 원칙입니다. (2.12.~2.28. 청구건 처리 후 전출처리 권장)
  • 간편결제(PAYCO/네이버페이): 영수증 청구 불가 → 조정점수로 특별검진비 부여 후 자율항목으로 사용(지난해 동일).
  • 퇴직자 건강검진: 재직기간 내 받은 검진만 청구 가능(예: 2.28 퇴직, 3.15 검진은 지급 불가).

1. 운영 일정 및 시스템 이용 안내

  • 복지시스템(포털) 개통: 2026년 2월 12일(목) 예정
  • 자율항목 청구 기간: 시스템 개통 이후 ~ 2026년 12월 9일(수)까지 (마감 이후 신청/지급 불가)
  • 접속 주소: 공무원연금공단 맞춤형복지 포털 (www.gwp.or.kr)
  • 개인별 확인: 로그인 후 본인 복지점수 확인 → 오류 시 즉시 변경 요청 / 지급 계좌 본인 명의 확인

인사변동 처리 일정(담당자 메모)

  • 3.1.자 전출자: 3.1. 이후 시스템 전출처리 (인사발령 “이전” 선처리 금지)
  • 권장 흐름: 시스템 개통(2.12.) 이후 2.12.~2.28. 청구건을 먼저 처리한 뒤, 3.1. 이후 전출처리

※ 기관별 운영/권한에 따라 처리 순서가 달라질 수 있으니, 관할 지침을 우선합니다.

실무 팁(담당자/교직원 공통)
  • 개통 초기에 “로그인/계좌/점수” 문의가 몰립니다 → 체크리스트로 안내하면 민원 소음이 줄어듭니다.
  • 마감 직전에는 증빙 누락이 폭증합니다 → 11월 말쯤 “사전 점검” 공지 1회 추천.

2. 복지점수 구성 및 배정 기준

구분 점수 비고
기본점수 1,200점 전 직원 일률 배정
근속점수 150~350점 근무연수에 따라 차등
가족점수(배우자) 100점
가족점수(직계존속) 1인당 100점
가족점수(직계비속) 첫째 100 / 둘째 200 / 셋째↑ 300점
전통시장·지역사랑 상품권 자율항목(보험료 제외) 금액의 10% 자동 전환 배정

※ “자율항목”에서 보험료를 제외한 사용액 기준으로 10%가 포인트로 배정됩니다.

3. 특별 지원 및 건강검진 안내

특별건강검진(2년 주기)
  • 2026년 대상: 짝수년도 출생자
  • 지원 한도: 30만 원(300점) 한도 내 실지급액
  • 배정 대상(휴직자 포함): 육아휴직자도 배정 대상입니다. (질병휴직 1년 이내, 공무상 질병휴직 3년 이내, 가사휴직 1년 이내 등 일반휴직자 포함)
  • 사용 범위: 본인 검진뿐 아니라, 본인이 비용을 지출한 경우 가족(배우자, 직계 존·비속 및 배우자의 직계존속) 건강검진에도 사용 가능
  • 신청 마감: 2026년 12월 9일(수)까지 조정점수 신청 + 영수증 청구 완료
  • 진행 순서: 검진 → 포털에서 조정점수 신청 → 승인 → 영수증 청구 (증빙서류 첨부)

※ 세부 기준은 “질의응답(건강검진편)” 운영 기준을 따릅니다.

건강검진 유의사항(담당자/퇴직자)
  • 퇴직일자 입력 상태에서는 조정점수 배정이 불가할 수 있습니다 → 퇴직처리 취소 후 조정점수 입력/승인 → 다시 퇴직처리
  • 월할배정 하지 않음: 신규임용/휴직/퇴직자 및 타시도 전출자도 (대상 요건 충족 시) 30만 원 전액 지원 가능
  • 퇴직자 청구 범위: 재직기간 내 사용한 검진비만 청구 가능 (예: 2.28. 퇴직 후 3.15. 검진은 지급 불가)
간편결제 유의사항(PAYCO, 네이버페이)
  • 간편결제 사용자는 영수증 청구가 불가합니다.
  • 특별검진비는 담당자가 조정점수로 특별검진비용을 부여한 뒤, 사용자가 자율항목으로 사용하는 방식입니다(지난해 동일).
  • 일반 결제 사용자는 증빙 첨부 후 영수증 청구가 가능합니다.
출산·난임·태아·산모 검진 지원
  • 출산 축하(자녀당 1회): 첫째 1,000점 / 둘째 3,000점 / 셋째↑ 5,000점
  • 신청 기한: 출생 연도 포함 3년 이내 신청 가능
  • 난임 지원: 재직 중 1회, 500점
  • 태아·산모 검진: 자녀 수 1명당 100점 배정

보고(유·초·중만 해당 / 원주교육지원청 행정과)

  • 출산축하·난임지원·태아·산모 검진 지원은 점수 배정뿐 아니라 현황 보고가 필요합니다.
  • 제출: 신청서(엑셀 서식) + 증빙서류(스캔본) 첨부하여 제출
  • 원본: 증빙서류 원본은 학교(기관) 자체 보관

4. 단체보험 및 항목 선택

  • 필수 항목: 생명·상해 보장보험은 의무 가입(보장금액 5천만/1억/2억 중 선택)
  • 선택 항목: 특정질병 진단비, 치아보험, 실손의료비 등은 개인 필요에 따라 선택
  • 실손보험 중복 확인: 개인 실손보험이 있으면 단체 실손보험 가입 중지 가능 → 중복 여부 확인 안내

5. 신분 변동 시 유의사항

  • 월할 계산: 신규 임용/퇴직/휴직 등 변동 발생 시 실제 근무한 달 기준으로 점수 월할
  • 퇴직 예정자: 정년·명예퇴직 예정자는 퇴직 전 점수 사용 권장 (퇴직 후 2개월 이내에만 청구 가능)
  • 이월·현금화 불가: 당해 연도 미사용 점수는 다음 연도 이월/금전 보상 불가 → 기한 내 전액 사용 독려
전보/전출 처리(강원도 내 이동 포함)
  • 강원도 내 기관 이동은 전출이 아니라 전보 처리입니다. (전출처리 X)
  • 만약 잘못 전보처리했을 경우, 전입 기관에서 인사변동 취소 처리로 되돌립니다.
  • 인사발령 시점 이전 선처리 금지: 3.1.자 전출자는 3.1. 이후 시스템 전출처리
  • 추가 메모: 2.12.~2.28. 사이 청구건을 처리한 뒤, 3.1. 이후 전출처리
퇴직 처리(담당자 메모)
  • 퇴직일자 입력: 마지막 근무일로 입력 후 퇴직 저장 (예: 2.28. 명퇴자 → 2.28 입력)
  • 퇴직 후 마감: 사용점수 확인 후, (잔액이 있어도) 퇴직 후 2개월 경과 시 퇴직 처리현황에서 “정산중” 클릭 → 팝업 하단 “정산완료”

※ 메뉴명/위치는 시스템 업데이트로 달라질 수 있습니다.

6. 결제 및 청구 방식

  • 카드 등록: 포털에 복지카드 등록 후 자동청구 또는 수동청구 중 선택
  • 사용 제한: 사행성 지출, 유흥비, 미용 관련 의료행위(성형/치열교정 등), 일반 상품권 구매 등은 사용 제한
  • 간편결제: PAYCO/네이버페이 사용자는 일부 항목(특별검진 등)에서 영수증 청구 불가 → 조정점수 부여 후 자율항목 사용
조정점수 승인 ≠ 지급 시작 (흐름 요약)
  1. 사용자: (검진/사유 발생 후) 증빙 첨부하여 조정점수 신청
  2. 담당자: 조정점수 승인 (점수 배정 상태)
  3. 사용자: 포털에서 영수증 청구 진행 (승인된 점수로 청구)
  4. 담당자: 승인관리에서 최종 승인 → 다음 달 급여일 지급

포인트: “승인”만 해놓고 “영수증 청구”가 안 되어 있으면, 지급이 안 됩니다.

영수증 대리청구(필요 시, 담당자)
  • 경로 예시: [복지점수 관리] → [청구관리] → [영수증 대리청구]
  • 대상자 검색 → 하단 [영수증 청구] 클릭 → 영수증 일자/금액 입력 후 저장
  • 복지항목은 예: ‘병의원진료’ 등으로 입력
  • 대리청구 후에도 승인관리에서 승인 처리를 완료해야 지급 대상에 포함됩니다.

※ 메뉴명은 시스템 버전에 따라 약간 다를 수 있습니다.

지급 시기 및 문의처
  • 지급 시기: 승인일의 다음 달 급여일 지급 (예: 2월 승인 → 3월 급여일 지급)
  • 지급 문의: 교원 → 유초등교육과 / 지방공무원 → 총무과
